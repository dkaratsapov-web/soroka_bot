# Проект: бот «лид-магнит за подписку» (Telegram + позже MAX)

Это контекст для Claude Code. Прочитай его целиком перед работой.

## Цель проекта
Бот выдаёт бесплатный гайд в обмен на подписку на канал. Нужен в Telegram, вторым
этапом — в мессенджере MAX. Без платных конструкторов: бот простой и типовой.

## Что настраивается в @BotFather, а не в коде
- Имя/username бота — при создании через `/newbot`.
- Описание — `/setdescription` (текст из карточки выше).
- «О боте» — `/setabouttext`.
- Аватар бота — `/setuserpic`.
- **Заставка** (картинка/гиф, видимая ДО нажатия Start) — Edit Bot → Edit Description Picture.

## Картинки
Различай два места:
1. **Заставка до Start** — ставится только в @BotFather (см. выше). Кодом не управляется.
2. **Картинка в первом сообщении бота** — поддержана в коде: положи файл рядом и укажи
   путь в `.env` через `WELCOME_IMAGE=welcome.jpg`. Пусто -> приветствие без картинки.

## Как запустить локально
1. `bash setup.sh`
2. Заполнить `.env`: только `BOT_TOKEN` (от @BotFather) обязателен; канал уже вписан.
   При наличии картинки приветствия — добавить `WELCOME_IMAGE=welcome.jpg`.
3. Сделать бота **администратором** канала @sorokavmarketinge.
4. `source venv/bin/activate && python leadmagnet_bot.py`
